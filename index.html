About MeasureFlow

MeasureFlow brings clarity to complex workflows with precision tools

Helping Measuring Technicians to focus on more complex and revenue-generating tasks by Automating reports, generating TSC, direct maps and google search,and a Manual tracker for job's points, status and time taken to measure.

Shortcuts:

Ctrl/⌘ + J → Search Web

Ctrl/⌘ + M → Open Google Maps

Ctrl/⌘ + K → Open ConnectExplorer

Ctrl/⌘ + D → Show Edit User + Download

Ctrl/⌘ + Shift + P → Mark All Jobs Passed (sets every job’s status to "Passed")

Ctrl/⌘ + Shift + Enter → Open Reject Job Form Directly

Ctrl/⌘ + Shift + X → Close All Measuring Tabs except for active one
